# جُمانُ الهُدَى

Landing page — React + Vite + TypeScript + Tailwind.

## Structure

```
├── public/
├── src/
│   ├── components/
│   ├── pages/
│   ├── data/
│   ├── config/
│   ├── App.tsx
│   ├── main.tsx
│   └── index.css
├── index.html
├── vite.config.ts
├── vercel.json
└── package.json
```

## Scripts

```bash
npm install
npm run dev
npm run build
```
